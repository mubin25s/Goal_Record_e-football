## 🛠️ Tech Stack
- **Frontend**: React, TypeScript, Vite, Vanilla CSS, Lucide Icons
- **Database & Auth**: Firebase Authentication & Cloud Firestore
- **File Storage**: Supabase Storage (`matches` bucket)
